## 블로그를 시작하며

개발하면서 배운 것들을 기록하고 공유하기 위해 블로그를 시작했습니다.

## 기술 스택

이 블로그는 다음 기술로 만들어졌습니다:

- **Astro** — 정적 사이트 생성
- **Tailwind CSS** — 스타일링
- **MDX** — 마크다운 + JSX
- **GitHub Pages** — 호스팅
- **GitHub Actions** — 자동 배포

## 코드 예시

```typescript
// site.config.ts에서 모든 문구를 관리합니다
export const SITE = {
  title: 'Ash-World',
  hero: {
    greeting: 'Hello,',
    name: 'ash-hun',
  },
};
```

앞으로 AI, 백엔드, 개발 경험 등 다양한 주제로 꾸준히 글을 올릴 예정입니다.
